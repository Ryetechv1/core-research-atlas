CORE Shapeshifting Research Atlas - Codex Build Brief

Current scope
-------------
The app is a local-first research atlas for source logging, file imports, an
in-app browser, team collaboration, document drafting, profiles, and memory
snapshots.

Removed scope
-------------
Do not rebuild the old assistant panel, standalone retrieval console, crawler
tools, automated research loop, or related backend routes. The app should stay
focused on browse, log, import, draft, review, and export workflows.

Preserve
--------
- Local gated sign-in for UserSeth, UserSemaj, UserKhiimori, plus read-only
  guest browsing.
- Source categories, species filters, evidence tiers, citation status, and
  source-detail previews.
- Google Programmable Search inside the Browser panel using CSE ID
  56f7592d1993141c3.
- Optional server-side Google Custom Search JSON through /api/search when a
  backend host provides GOOGLE_CUSTOM_SEARCH_API_KEY.
- File import for research files and ChatGPT export structures.
- Team posts, updates, links, recommendations, add/reject handling, and
  cross-tab sync.
- Collaboration documents with create, edit, browser-lead insertion, import,
  export, and print-to-PDF.
- Profiles for the three users and guest read-only protections.
- Internal memory snapshots and exportable memory-bank JSON.
- Uploaded PDF/reference metadata as background source leads only.

3. Stronger document parsing and citation-review tooling.
4. Public deployment that does not depend on local tunnels.
