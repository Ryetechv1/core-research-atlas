Place a licensed ITC Benguiat font file in this folder to make the hosted app render Benguiat on every device.

Supported filenames:

- ITC-Benguiat.woff2
- ITC-Benguiat.woff
- ITC-Benguiat.ttf
- ITC-Benguiat.otf
- Benguiat.woff2
- Benguiat.woff
- Benguiat.ttf
- Benguiat.otf

The app also checks common locally installed font names, but mobile browsers and other users will only see exact ITC Benguiat if the font is installed on their device or a licensed font file is hosted here.
